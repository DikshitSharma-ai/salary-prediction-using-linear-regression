# Employee Salary Prediction using Multiple Linear Regression(ML)

## Project Overview

This is a beginner machine learning project built using Python and Scikit-learn.

The goal of this project is to predict an employee's salary based on:

- Experience
- Hours Worked Per Day

A Multiple Linear Regression model is used for prediction.

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn

---

## Workflow

1. Load dataset
2. Select features and target
3. Split data into training and testing sets
4. Train Linear Regression model
5. Make predictions
6. Evaluate model using:
   - MAE
   - MSE
   - R2 Score
7. Visualize results using Matplotlib

---

## Features Used

- Experience
- Hours_Worked

## Target

- Salary

---

## Evaluation Metrics

The model performance is measured using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R2 Score

---

## Visualizations

The project includes:

- Experience vs Salary Scatter Plot
- Actual Salary vs Predicted Salary Plot
